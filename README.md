# ruby-push
Ruby script utilizing the Pushover API

## Package Requirements (Linux OS)
- ruby v3 or greater
- gem v3 or greater
- net-http (default: 0.2.2)
- base64 (0.2.0, default: 0.1.1)

## Usage
`ruby push.rb "message goes here`

## Config.yaml File
- Set default parameters to your notifications
- Easy configuration
- Expandable
